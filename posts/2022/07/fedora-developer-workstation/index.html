Fedora Developer Workstation

Photo by Micaela Alejandro

Summary

As part of my migration from Debian to Fedora, I’ve put together this brief guide to turning a fresh install of Fedora Workstation into a comfortable development environment geared towards getting things done. I’ve chosen packages and tools to support working with Python, TypeScript, Node, Rust, Go and even a little bit of Java.

I try to keep things as tidy as possible with Flatpak and Docker, while being reasonable about creature comforts like media codecs and graphics drivers. ðŸ˜„ This is far from being a stripped down and ‘bloat-free’ Arch installation (more on that next time!), however it is stable and boring, and when you’re trying to get stuff done, that counts for a lot.

Get Updates

# Check for updates
dnf check-update

# Install updates
sudo dnf upgrade

Enable RPMFusion and Flathub

# Flathub Repos
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

# RPMFusion Free Repos
sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm

# RPMFusion Nonfree Repos
sudo dnf install https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Install Multimedia Codecs:

sudo dnf install gstreamer1-plugins-{bad-\*,good-\*,base} gstreamer1-plugin-openh264
sudo dnf install lame\* --exclude=lame-devel
sudo dnf group upgrade --with-optional Multimedia

Install Nvidia Drivers:

# Add NVIDIA drivers to the system
sudo dnf install akmod-nvidia

# Optional: Add CUDA support
sudo dnf install xorg-x11-drv-nvidia-cuda

# Check if everything works, wait a few minutes
modinfo -F version nvidia

Note: The Zoom flatpak looks a little crusty, I think I’ll just use the browser instead. The Chrome flatpak isn’t quite there yet either. I’ve also opted not to use VS Code as a flatpak.

VSCode Flatpak Extensions:

If you’re stubborn, like me, and you want to give the VS Code Flatpak a go, you’ll need to install and enable the neccesary SDKs after installing VS Code from Flathub. Since my focus here is on GO, Rust, Node, and Java, you’ll need to install the extensions below. Python is already catered for.

# Install VS Code
flatpak install flathub com.visualstudio.code

# Install SDKs
flatpak install flathub org.freedesktop.Sdk.Extension.golang org.freedesktop.Sdk.Extension.node16 org.freedesktop.Sdk.Extension.rust-stable org.freedesktop.Sdk.Extension.openjdk17

# Enable SDKs
FLATPAK_ENABLE_SDK_EXT=rust-stable,golang,node16,openjdk17 flatpak run com.visualstudio.code

Google Chrome

# Check to make sure you've got third party repositories enabled.
sudo dnf install fedora-workstation-repositories

# Enable the Google Chrome repository
sudo dnf config-manager --set-enabled google-chrome

# Install Google Chrome
sudo dnf install google-chrome-stable

Docker:

# Extend DNF with Core Plugins
sudo dnf -y install dnf-plugins-core

# Add the Docker repository
sudo dnf config-manager --add-repo https://download.docker.com/linux/fedora/docker-ce.repo

# Install Docker-Engine and Compose
sudo dnf install docker-ce docker-ce-cli containerd.io docker-compose-plugin

# Start Docker
sudo systemctl start docker

# Make sure Docker starts on boot
sudo systemctl enable docker.service
sudo systemctl enable containerd.service

Docker Images:

# Pull docker images from the Docker Hub
for img in python node mongo-express postgres dpage/pgadmin4 redis cassandra nginx mysql; do sudo docker pull $img; done

Note: Feel free to skip this step if you don’t need any images yet.

Additional Packages:

# Install packages
sudo dnf install pipenv nodejs hugo rust cargo golang java-latest-openjdk-devel.x86_64

Note: pipenv pulls in pip and virtualenv, node pulls in npm.

# Add the VS Code repository
sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
sudo sh -c 'echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" > /etc/yum.repos.d/vscode.repo'

# Update and install VS Code
dnf check-update
sudo dnf install code
